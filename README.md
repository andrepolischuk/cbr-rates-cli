# cbr-rates-cli [![Build Status][travis-image]][travis-url]

> Get currency rates from Central Bank of Russia

[cbr-rates][cbr-rates] - API for this module

![](screenshot.png)

## Install

```sh
npm install --global cbr-rates-cli
```

## Usage

```sh
cbr-rates --help

  Usage
    cbr-rates [...id] [date]

  Examples
    cbr-rates
    cbr-rates usd eur
    cbr-rates 2014.5.12
```
