# cbr-rates-cli

  > Get currency rates from Central Bank of Russia

  [cbr-rates][cbr-rates] - API for this module

## Install

```sh
npm install --global cbr-rates-cli
```

## Usage

```sh
cbr-rates --help

  Usage
    cbr-rates [date]

  Examples
    cbr-rates
    cbr-rates 2014.5.12
```
